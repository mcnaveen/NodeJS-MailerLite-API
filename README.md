## Usage

- Clone or Download the repo

```sh
git clone https://github.com/mcnaveen/NodeJS-MailerLite-API.git
cd NodeJS-MailerLite-API
npm install
```

- create `.env` file and add Mailerlite API key and group ID

```sh
touch .env
```

- Run the Script

```sh
npm start
```
